# 04 — โครงหน้าจอและ Flow การใช้งาน (UI Flows)

## โครงหน้าจอรวม

```
┌──────────────────────────────────────────────────────────────┐
│ ผู้ทำรายการ: [ แม่ ▼ ]            Home Stock Manager           │  ← แถบบน (เลือกผู้ใช้)
├──────────────────────────────────────────────────────────────┤
│ [ รายการของ ] [ รับเข้า/เบิกออก ] [ ประวัติ ] [ แจ้งเตือน ] [ รายงาน ] │  ← แท็บ
├──────────────────────────────────────────────────────────────┤
│                                                                │
│                    (เนื้อหาของแท็บที่เลือก)                       │
│                                                                │
└──────────────────────────────────────────────────────────────┘
```

แถบบนมี dropdown เลือกชื่อผู้ทำรายการ ค่าที่เลือกถูกใช้บันทึกใน transaction
ของแท็บรับเข้า/เบิกออก

---

## แท็บ 1: รายการของ (Items)
- ช่องค้นหา (ชื่อ/บาร์โค้ด) + dropdown กรองตามหมวดหมู่/ตำแหน่ง
- ปุ่ม **+ เพิ่มของ** เปิดฟอร์ม (ชื่อ, หมวด, ตำแหน่ง, หน่วย, จำนวนเริ่มต้น, จำนวนขั้นต่ำ, บาร์โค้ด, วันหมดอายุ)
- ตารางแสดง: ชื่อ | หมวด | ตำแหน่ง | คงเหลือ | หน่วย | ขั้นต่ำ | วันหมดอายุ | [แก้ไข] [ลบ]
- **ไฮไลต์สี:** แถวของใกล้หมด (เหลือ ≤ ขั้นต่ำ) = สีเหลือง/ส้ม, ของหมดอายุ/ใกล้หมดอายุ = สีแดง

**Flow เพิ่มของ:** คลิก + เพิ่มของ → กรอกฟอร์ม → บันทึก → รายการขึ้นในตารางทันที

---

## แท็บ 2: รับเข้า/เบิกออก (Check-in / Check-out)
- ช่อง **บาร์โค้ด/ค้นหา** (โฟกัสอัตโนมัติ — สแกนเครื่อง USB ได้เลย) → เจอของก็เลือกให้
- หรือเลือกของจาก dropdown
- แสดงจำนวนคงเหลือปัจจุบันของรายการที่เลือก
- ช่อง **จำนวน** + ช่อง **หมายเหตุ**
- ปุ่ม **รับเข้า (Check-in)** = เพิ่มจำนวน | ปุ่ม **เบิกออก (Check-out)** = ลดจำนวน
- ถ้าเบิกเกินคงเหลือ → ขึ้นข้อความเตือนสีแดง ไม่บันทึก

**Flow เบิกออก:** สแกน/เลือกของ → ใส่จำนวน → กดเบิกออก →
ตรวจจำนวนพอไหม → ถ้าพอ: ลดจำนวน + บันทึกประวัติ + แจ้งสำเร็จ → ถ้าไม่พอ: เตือน

---

## แท็บ 3: ประวัติ (History)
- ตัวกรอง: รายการของ (dropdown), ผู้ใช้ (dropdown), ช่วงวันที่
- ตารางแสดง: เวลา | ของ | ประเภท (รับเข้า/เบิกออก) | จำนวน | ผู้ทำรายการ | หมายเหตุ
- เรียงเวลาล่าสุดอยู่บน

---

## แท็บ 4: แจ้งเตือน (Alerts)
- กล่อง **ของใกล้หมด:** รายการที่ คงเหลือ ≤ ขั้นต่ำ (ชื่อ, คงเหลือ/ขั้นต่ำ)
- กล่อง **ของใกล้/หมดอายุ:** รายการที่วันหมดอายุภายใน N วัน (ตั้งค่า N ได้, ค่าเริ่มต้น 30)
  แสดงของที่หมดอายุแล้วด้วย (สีแดงเข้ม)
- ช่องปรับจำนวนวันเตือนล่วงหน้า

---

## แท็บ 5: รายงาน (Reports)
- การ์ดสรุป: จำนวนรายการทั้งหมด, จำนวนของใกล้หมด, จำนวนของใกล้หมดอายุ
- ปุ่ม **Export รายการของ (CSV)** → เปิด dialog เลือกที่เซฟ
- ปุ่ม **Export ประวัติ (CSV)** → เปิด dialog เลือกที่เซฟ
- ไฟล์ CSV เข้ารหัส UTF-8 เปิดอ่านภาษาไทยได้
